# AI Compiler

An AI Engineering internship project exploring the design and implementation of an AI Compiler.

## Status

**Phase 1 — Repository Scaffold.** No business logic has been implemented. This phase only establishes project structure, tooling, and a minimal health-check endpoint to confirm the stack is wired correctly.

## Stack

- **Backend:** Python 3.11, FastAPI, Pydantic
- **Frontend:** React + Vite, TypeScript
- **Testing:** pytest (backend), Vitest (frontend)

## Structure

```
backend/      FastAPI application
frontend/     React + Vite + TypeScript application
docs/         Project documentation
tests/        Backend and frontend tests
evaluation/   Evaluation scripts and datasets (future phases)
```

See [docs/setup.md](docs/setup.md) for setup instructions.
